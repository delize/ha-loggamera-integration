Loggamera Integration for Home Assistant

This documentation helps you set up and use the Loggamera integration with Home Assistant.

Installation

Manual Installation

  1. Download the integration files from the GitHub repository.
  2. Copy the loggamera folder to your Home Assistant configuration directory under custom_components/.
  3. Restart Home Assistant.

HACS Installation

  1. In HACS, go to "Integrations".
  2. Click the three dots in the upper right corner and select "Custom repositories".
  3. Add the repository URL and select "Integration" as the category.
  4. Click "Add".
  5. Search for "Loggamera" and install it.
  6. Restart Home Assistant.

Configuration

  1. In Home Assistant, go to Configuration > Devices & Services.
  2. Click the "+ Add Integration" button in the bottom right.
  3. Search for "Loggamera" and select it.
  4. Enter your API key when prompted.
  5. Click "Submit".
Note: You can find your API key in your Loggamera account settings.

Entities

The integration will automatically create entities based on your Loggamera devices:

Power Meters

Each power meter will create entities for:

  • Power consumption (kW)
  • Total energy usage (kWh)

Water Meters

Each water meter will create entities for:

  • Water consumption (m³)
  • Water usage since midnight

Room Sensors

Room sensors will create entities for:

  • Temperature (°C)
  • Humidity (if available)

Cooling Units & Heat Pumps

These will create various sensors based on available data:

  • Temperatures at various measurement points
  • Operating modes
  • Alarm status

Services

The integration provides the following services:

execute_scenario

Execute a scenario defined in your Loggamera account.

Service: loggamera.execute_scenario

Parameters:

Parameter Required Description
scenario_id Yes ID of the scenario to execute
duration_minutes No Duration in minutes for the scenario (if applicable)
entry_id No Entry ID of your Loggamera integration (only needed if you have multiple instances)

Example YAML:

service: loggamera.execute_scenario
data:
  scenario_id: 12345
  duration_minutes: 60

Troubleshooting

Common Issues

  • Connection errors: Check your API key and internet connection.
  • Missing devices: Ensure your devices are properly set up in Loggamera.
  • No data: The API may be temporarily unavailable. Check the Loggamera status page.

Logs

To enable debug logs, add the following to your configuration.yaml:

logger:
  default: warning
  logs:
    custom_components.loggamera: debug
